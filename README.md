# Automated Pipeline for Cabbage plugins

⚠️ Still a WIP

Automated steps:

- Install Cabbage and Csound in an Azure Pipeline
- Export VST, VST3, AU
- Relink Csound
- Codesign the VST, VST3 and AU exports
- Create installers (including Presets/ folder)
- Codesign and notarize the installer
